# Exploratory Data Analysis (EDA) Project

## Objective

Perform Exploratory Data Analysis on an E-commerce Sales Dataset using Python.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL

## Project Steps

1. Loaded the dataset
2. Explored the dataset
3. Generated statistical summary
4. Analyzed product sales
5. Analyzed payment methods
6. Created visualizations:
   - Product Sales Bar Chart
   - Payment Method Pie Chart
   - Histogram
   - Box Plot
   - Correlation Heatmap

## Key Insights

- Product sales distribution was analyzed.
- Payment methods were almost equally distributed.
- Total Price distribution was visualized.
- Outliers were identified using a box plot.
- Correlation between numerical features was analyzed using a heatmap.
