# DecodeLabs Internship - Task 3

# SQL Data Analysis using MySQL Workbench

## 📌 Project Overview

This project was completed as part of the **DecodeLabs Data Analyst Internship**.

The objective of this project is to perform SQL data analysis on an Orders dataset using MySQL Workbench. Various SQL queries were executed to extract meaningful business insights from customer order data.

---

## 🛠 Technologies Used

- MySQL Workbench
- SQL
- Microsoft Excel

---

## 📂 Dataset Information

The dataset contains **1200 customer order records** with the following columns:

- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- ShippingAddress
- PaymentMethod
- OrderStatus
- TrackingNumber
- ItemsInCart
- CouponCode
- ReferralSource
- TotalPrice
